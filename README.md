 AI Chatbot Project

Overview
This project is a rule-based chatbot developed using Python. It simulates human-like conversations and responds to user queries using predefined rules and conditions.

Features
- Interactive text-based conversation
- User-friendly responses
- Input validation
- Greeting and farewell messages
- Continuous chat support

 Key Highlights
This chatbot processes user inputs, identifies keywords, and generates appropriate responses. The project demonstrates Python fundamentals such as loops, conditional statements, functions, and string handling while providing an engaging conversational experience.

 Technologies Used
- Python 3
- Functions
- Loops
- Conditional Statements
- String Manipulation
